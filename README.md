## Technical Report

The file `DICE-TechnicalReport.pdf` in this directory is the technical report accompanying our paper “DICE: Decentralized Incentive-driven Federated Learning for Efficient & Secure Customization of Foundation Models in Web3”. It contains detailed proofs, additional experimental results, and further discussions that complement the main text.

## Code and Data

The complete source code and datasets used in this paper are currently being prepared for public release. They will be made available upon publication of the paper. We are committed to supporting reproducibility and will provide detailed documentation at that time.
